## 常用基础术语 {#common-basics}

| 术语                                                                    | 核心定义                                                             | 备注                                       |
| ----------------------------------------------------------------------- | -------------------------------------------------------------------- | ------------------------------------------ |
| [多意识体系统（MPS）](entries/Multiple_Personality_System.md)           | 指在同一身体内存在两个及以上、能够持续运作的意识单元（成员）的整体。 | 全站概念基础，延伸出前台、后台等术语。     |
| [系统（System）](entries/System.md)                                     | 拥有多个成员并共享资源、目标与身份的主体。                           | 与“多意识体系统”同义，常用于社群自称。     |
| [成员（Alter）](entries/Alter.md)                                       | 具有独立身份感、记忆或功能侧重的意识单元。                           | DSM-5-TR 语境中称为“身份状态”。            |
| [切换（Switch）](entries/Switch.md)                                     | 前台控制权由一位成员过渡至另一位成员的过程。                         | 可能伴随感官、行为或记忆的短暂调整。       |
| [前台（Front / Fronting）](entries/Front-Fronting.md)                   | 当前与外界互动并控制身体的成员或成员组合。                           | 与共前台、共意识密切相关。                 |
| [后台（Back / Being Back）](entries/Back-Being-Back.md)                 | 未执掌前台、处于内在活动或休息状态的成员。                           | 亦可描述暂时离开身体的体验。               |
| [共前台（Co-fronting）](entries/Co-Fronting.md)                         | 两位及以上成员共同掌控前台并协作执行任务。                           | 关注行为层面的协同。                       |
| [共意识（Co-consciousness）](entries/Co-Consciousness.md)               | 多位成员同时保持觉察并共享当前体验。                                 | 常与记忆连通度、团队合作程度相关。         |
| [内部空间（Headspace / Inner World）](entries/Headspace-Inner-World.md) | 成员在内在世界中互动、生活或储存记忆的心象环境。                     | 结构因系统而异，可包含房间、景观或象征物。 |
| [触发（Trigger）](entries/Trigger.md)                                   | 引发强烈情绪、记忆或解离反应的内外部线索。                           | 创伤知情实践中需提前识别并建立应对计划。   |

## 系统运作与体验 {#system-operations}

| 术语                                                                        | 核心定义                                                   | 备注                                         |
| --------------------------------------------------------------------------- | ---------------------------------------------------------- | -------------------------------------------- |
| [并行（Parallelism）](entries/Parallelism.md)                               | 多位成员在同一时间维持独立意识流或任务处理。               | 可能导致注意力分配与记忆记录的差异。         |
| [内在沟通（Internal Communication）](entries/Internal-Communication.md)     | 成员之间传递信息、协作决策的方式与工具。                   | 包含会议、日志、信号系统等实践。             |
| [混合（Blending）](entries/Blending.md)                                     | 成员间的感受、记忆或表达出现叠加，边界暂时模糊。           | 常见于快速切换或高度压力情境。               |
| [记忆屏蔽（Memory Shielding）](entries/Memory-Shielding.md)                 | 为保护系统功能而主动隔离特定记忆或情绪信息。               | 与创伤史、角色分工紧密相关。                 |
| [子系统（Subsystem）](entries/Subsystem.md)                                 | 系统内部进一步分化出的成员子群或结构。                     | 需要额外的协调机制以保持一致。               |
| [自动驾驶（Autopilot）](entries/Autopilot.md)                               | 成员以低觉察度执行习惯化任务的状态。                       | 可视为应对疲劳或过载的保护策略。             |
| [功能性解离（Functional Dissociation）](entries/Functional-Dissociation.md) | 为完成复杂任务而产生的适应性注意分离或角色分配。           | 与病理性解离区分，强调功能性。               |
| [外部投射（External Projection）](entries/External-Projection.md)           | 将内部成员的形象或声音投射到外部媒介以辅助沟通。           | 常见于虚拟形象、角色扮演或艺术创作。         |
| [解离性遗忘（Dissociative Amnesia）](entries/Dissociative-Amnesia-DA.md)    | 对特定事件、阶段或个人信息出现选择性记忆缺失。             | 需与普通遗忘区分，可能伴随切换或触发反应。   |
| [非我感（Not-me Feeling）](entries/Not-Me-Feeling.md)                       | 对其他成员的行为、记忆或情绪产生"那不是我"的主观区隔体验。 | 用于识别成员边界的核心机制，与人格解体不同。 |

## 系统角色与功能 {#system-roles}

| 术语                                                              | 核心定义                                                     | 备注                             |
| ----------------------------------------------------------------- | ------------------------------------------------------------ | -------------------------------- |
| [宿主（Host）](entries/Host.md)                                   | 在多数时间负责日常生活、对外沟通的成员。                     | 并非所有系统都存在单一宿主。     |
| [保护者（Protector）](entries/Protector.md)                       | 负责安全评估、危机应对或边界维护的成员。                     | 可细分为情绪型、防御型等。       |
| [照护者（Caregiver）](entries/Caregiver.md)                       | 专注内部支持、照顾年幼成员或日常维持的角色。                 | 常与养育经验或专业技能相关。     |
| [守门人（Gatekeeper）](entries/Gatekeeper.md)                     | 管理切换流程、记忆访问或内部空间出入的成员。                 | 在复杂系统中承担协调与审查职责。 |
| [迫害者（Persecutor）](entries/Persecutor.md)                     | 通过攻击性或自我惩罚行为表达警讯的成员。                     | 需以创伤知情视角理解其保护意图。 |
| [记忆保管者（Memory Holder）](entries/Memory-Holder.md)           | 负责储存或整理特定记忆片段的成员。                           | 常协助事件回忆与时间线修复。     |
| [儿童人格（Little）](entries/Child-Alter.md)                      | 年龄感知较低、保有童年记忆或需求的成员（亦称“小孩意识体”）。 | 支持时需强调安全感与稳定环境。   |
| [青少年成员（Teen）](entries/Teen-Alter.md)                       | 具有青少年视角与需求的成员。                                 | 在社交、自主需求上提供独特观点。 |
| [执行者（Performer / Executive）](entries/Performer-Executive.md) | 擅长在高压情境保持表现或完成任务的成员。                     | 常用于工作、演讲、应急等场景。   |
| [初始（Original）](entries/Original.md)                           | 被系统视为最早出现或最贴近出生身份的成员。                   | 不一定与宿主重合，概念存在争议。 |

## 实践方法与自助工具 {#practice-support}

| 术语                                                                            | 核心定义                                           | 备注                                 |
| ------------------------------------------------------------------------------- | -------------------------------------------------- | ------------------------------------ |
| [接地（Grounding）](entries/Grounding.md)                                       | 透过感官刺激、动作或呼吸帮助回到当下。             | 用于缓解解离、焦虑或惊恐反应。       |
| [正念（Mindfulness）](entries/Mindfulness.md)                                   | 以非评判态度觉察当下经验的训练方式。               | 可提升成员协作与情绪调节能力。       |
| [冥想（Meditation）](entries/Meditation.md)                                     | 透过专注练习稳定注意力、情绪与身体觉察。           | 需评估触发风险，循序渐进地开展。     |
| [视觉化（Visualization / Imagination）](entries/Visualization-Imagination.md)   | 主动想象内部空间、沟通场景或功能分配。             | 与内部会议、图帕培育等实践相关。     |
| [意识修改（Consciousness Modification）](entries/Consciousness-Modification.md) | 调整成员觉察、边界或前台访问方式的练习。           | 可支持切换管理与角色协作。           |
| [整合（Integration）](entries/Integration.md)                                   | 强化成员协作、记忆连结与功能稳定的长期目标。       | 不等同于“消融”成员，强调合作。       |
| [三阶段创伤治疗模型](entries/Three-Phase-Trauma-Treatment.md)                   | 包含安全稳定、创伤加工、整合重建三阶段的治疗流程。 | 创伤知情干预的经典框架。             |
| [感官调节策略](entries/Sensory-Regulation-Strategies.md)                        | 调整感官输入以维持功能与舒适度的方法。             | 适用于触发、自闭谱及神经多样性需求。 |
| [创伤疗愈指南](entries/Trauma-Healing-Guide.md)                                 | 汇整创伤评估、支持网络与自助流程的实务指南。       | 提供自我照护与专业协作建议。         |
| [自我照护工具包](entries/Self-Care-Toolkit.md)                                  | 规划日常照护与危机应对资源的工具集合。             | 可用于成员交接与应急检查表。         |

## 诊断与临床术语 {#clinical-terms}

| 术语                                                                                      | 核心定义                                                | 备注                                 |
| ----------------------------------------------------------------------------------------- | ------------------------------------------------------- | ------------------------------------ |
| [解离性身份障碍（DID）](entries/DID.md)                                                   | 以多个身份状态与记忆缺失为特征的解离性障碍。            | 需结合创伤史、功能影响进行诊断。     |
| [其他特定解离性障碍（OSDD）](entries/OSDD.md)                                             | 未完全符合 DID 标准但具有显著解离症状的诊断范畴。       | 常见子类型包含 OSDD-1a、OSDD-1b。    |
| [解离性障碍谱系](entries/Dissociative-Disorders.md)                                       | 包含解离性遗忘、人格解体-现实解体障碍等在内的障碍群。   | 评估时需关注记忆、身份与现实感。     |
| [创伤后应激障碍（PTSD）](entries/PTSD.md)                                                 | 经历创伤后出现侵入、回避、负性认知与唤醒症状。          | 多意识体成员可能呈现差异化反应。     |
| [复杂性创伤后应激障碍（CPTSD）](entries/CPTSD.md)                                         | ICD-11 对长期虐待、囚禁等创伤的诊断，强调自体功能受损。 | 包含情绪调节、负性自我观与关系困难。 |
| [急性应激障碍（ASD）](entries/Acute-Stress-Disorder-ASD.md)                               | 创伤后 3 天至 1 个月内出现的解离、焦虑与侵入性症状。    | 若持续超过 1 个月需评估 PTSD。       |
| [人格解体-现实解体障碍（DPDR）](entries/Depersonalization-Derealization-Disorder-DPDR.md) | 长期体验自我感脱离或现实感淡化的症状群。                | 可独立存在或与其他解离障碍共病。     |
| [功能性神经症状障碍（FND）](entries/Conversion-Disorder-FND.md)                           | 出现神经症状但缺乏一致的器质性病因。                    | 需要跨学科评估与创伤知情复健。       |
| [创伤](entries/Trauma.md)                                                                 | 超出个体即时应对能力的威胁性经历及其影响。              | 创伤知情框架强调安全、赋权与合作。   |

## 理论模型与研究框架 {#theoretical-models}

| 术语                                                                    | 核心定义                                             | 备注                               |
| ----------------------------------------------------------------------- | ---------------------------------------------------- | ---------------------------------- |
| [结构性解离理论（SDT）](entries/Structural-Dissociation-Theory.md)      | 以 ANP/EP 模型解释创伤导致的心理结构分化。           | 指导创伤治疗与系统理解的重要理论。 |
| [ANP / EP 模型](entries/Apparently-Normal-Part-Emotional-Part-Model.md) | 将人格分为“表面正常部分”与“情绪部分”，阐释创伤反应。 | 属于结构性解离理论核心概念。       |
| [埃蒙加德分类](entries/Emmengard-Classification.md)                     | 社群提出的系统来源分类，如创造型、自发型、混合型等。 | 与图帕、系魂等实践讨论关联密切。   |
| [依恋理论](entries/Attachment-Theory.md)                                | 研究早期照料关系对安全感与情绪调节的影响。           | 用于分析系统内部信任与合作。       |
| [自我决定理论（SDT）](entries/Self-Determination-Theory.md)             | 强调自主性、胜任感、连结感三个基本动机需求。         | 可指导目标设定与支持策略。         |
| [社会认知理论](entries/Social-Cognitive-Theory.md)                      | 关注观察学习与自我效能在行为形成中的作用。           | 解释成员间的榜样与反馈机制。       |
| [动机理论](entries/Motivation-Theories.md)                              | 汇整需要层次、强化、期待等经典动机模型。             | 帮助设计内部激励与任务分配。       |
| [神经多样性框架](entries/Neurodiversity.md)                             | 主张神经差异是人类多样性的自然组成，倡导合理便利。   | 与自闭谱、ADHD 等议题交叉。        |

## 文化与社群用语 {#cultural-terms}

| 术语                                                                   | 核心定义                                                 | 备注                                 |
| ---------------------------------------------------------------------- | -------------------------------------------------------- | ------------------------------------ |
| [图帕（Tulpa）](entries/Tulpa.md)                                      | 通过有意想象与练习培育出的自主意识伙伴。                 | 与多意识体系统存在交集但成因不同。   |
| [图帕语（Tulpish）](entries/Tulpish.md)                                | 图帕与宿主之间发展出的独特沟通方式。                     | 可包含语言、意象或感官信号。         |
| [系魂（Soulbond）](entries/Soulbond.md)                                | 与虚构角色或外部实体形成稳定互惠联结的成员或关系。       | 在弦羽生态位等理论中常见。           |
| [系魂型系统](entries/Soul-Linked-Systems-Xianyu.md)                    | 弦羽理论提出的生态位分类之一，强调与外部意识的稳定连接。 | 常结合灵性、叙事或创作体验。         |
| [东方 Project 图帕社群](entries/Touhou-Tulpa-Fandom.md)                | 探讨东方 Project 同人圈内的图帕实践与社群文化。          | 展示亚文化对多意识体概念的应用。     |
| [《黑客军团》DID 叙事](entries/Mr-Robot-DID-Narrative.md)              | 分析剧集《黑客军团》中多意识体元素的表现手法。           | 可用于讨论主流媒体再现的误区。       |
| [《分裂》（2016）形象解析](entries/Split-2016-DID-Representation.md)   | 评估电影《Split》中对解离性身份障碍的描绘。              | 常被引用于公众误解的澄清。           |
| [《双面人》与身份分裂](entries/Dostoevsky-The-Double-Self-Division.md) | 解析陀思妥耶夫斯基作品中的身份裂解主题。                 | 关注文学作品对多重自我的诠释。       |
| [《三重人格》原型](entries/Three-Faces-Of-Eve-1957-Dissociation.md)    | 回顾电影《The Three Faces of Eve》的文化影响。           | 是早期大众对多重人格印象的重要来源。 |
| [文化与媒体指南](entries/Cultural-Media-Guide.md)                      | 汇总影视、文学、动画等对多意识体的再现案例。             | 提供更多延伸阅读与批判视角。         |
