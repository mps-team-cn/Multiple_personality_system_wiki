## 统一写作模板（复制即用）

```yaml
---
tags: [诊断与临床]
topic: 诊断与临床
title: <中文名>（<英文名/缩写>）
updated: 2025-10-16
status: draft   # draft | review | done
aliases: []     # 旧称/误写/历史术语（可选）
---
```

***首屏一句话定位**（≤120 字，回答“是什么/何时就医/何时急诊”）
***诊断要点**（核心症状、排除与阈值）
***ICD-11 对照**（若适用）
***鉴别诊断**（3–5 条，举例 + 关键分辨点）
***共病评估**（常见共病与筛查要点）
***治疗管理**（首选治疗、转诊与安全计划）
***互链**（至少 3 条同主题核心条目）

## 验收清单（Definition of Done）

| 类别   | 要求                                                   |

| 元数据 | `status`、`updated` 存在；`title` 格式规范             |
