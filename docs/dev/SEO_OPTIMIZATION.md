• # 关键词分析结论

  ## 1. 关键发现

  这批词反映出的站点现状很明确：站内已经具备较强的“正文深度”，但 SEO 层面的主要瓶颈不是没内容，而是“入口词设计不够贴近搜索语义”。很多高价值词已经有对应词条，但标题、摘
  要、同义词、对照页和导航页没有把搜索意图吃满。

  最值得投入的主题簇有 5 个：

  1. DID / OSDD / PDID / 多重人格 / 解离性人格障碍
  2. Tulpa / 多意识体 / 系魂 / 幻想伙伴 / 社群入口
  3. SAS / SDS / DES-II / MID-60 / SDQ-20 / SCL-90 / DSM-5量表
  4. 闪回 / 侵入性思维 / 侵入性记忆 / 情感闪回 / 躯体闪回
  5. 自我决定理论 / PTG / 社会认知理论 / 人格面具

  某些词排名不差但 CTR 很低，核心原因不是“没排名”，而是“标题和用户问题没有正面对齐”：

  - DID / MDD / PTG / ASD 这类缩写天然歧义高，用户想看的是“是什么意思/是什么病/和谁区别”，而不是纯学术标题。
  - 解离性人格障碍 这类误称有流量，但页面如果只用标准术语“解离性身份障碍”，CTR 会明显被拖低。
  - 闪回、人格面具、自我决定理论 这类词用户普遍要的是“快速解释 + 区别 + 应用”，不是长学术导论。
  - 量表词 CTR 普遍高，说明“工具型意图”与你的站点匹配度很高，应该继续模板化和聚合化。

  需要单独指出的临床误称/旧称/非标准术语：

  - 解离性人格障碍、解离型人格障碍、人格解离障碍：都不是标准诊断名称，应统一指向 解离性身份障碍（DID）
  - 多重人格、双重人格、split personality：旧称/俗称，应明确标注“非标准术语”
  - OSDD-1a / OSDD-1b：社群经验标签，不是 DSM/ICD 正式亚型
  - 多意识体综合征：非标准术语，不建议作为主标题
  - SAS抑郁量表：明显混淆，应并入 SDS
  - ASD：至少存在 孤独症谱系障碍 与 急性应激障碍 双重歧义，必须做导航页

  ## 3. 重点词条优化建议

  ### 1. DID 主词条

  推荐标题：DID 是什么？解离性身份障碍、旧称“多重人格”与常见误称说明
  推荐 description：解释 DID（解离性身份障碍）的诊断要点、常见表现、与 OSDD/PDID 的区别，并澄清“解离性人格障碍”“多重人格”等旧称和误称。
  推荐 synonyms：DID、解离性身份障碍、解离性人格障碍、解离型人格障碍、人格解离障碍、多重人格、split personality、did是什么病
  推荐 H2 结构：DID是什么、诊断标准、常见症状、与OSDD/PDID区别、与精神分裂症/BPD/DPDR区别、量表能否诊断、何时求助
  是否需要对照页/导航页：需要，必须新增 DID / OSDD / PDID 区别
  需要补的鉴别诊断或相关条目：OSDD、PDID、DPDR、解离性失忆、精神分裂症、边缘型人格障碍、多重人格（旧称）

  ### 2. 自我决定理论

  推荐标题：自我决定理论（SDT）：三种基本心理需求、动机连续体与实际应用
  推荐 description：用中文解释 SDT 是什么、谁提出、三种基本心理需求是什么，以及它如何应用于教育、治疗、组织管理与自我激励。
  推荐 synonyms：自我决定理论、SDT、self-determination theory、德西自我决定理论、自主性动机理论、三个基本心理需求
  推荐 H2 结构：SDT是什么、谁提出的、三种基本心理需求、动机连续体、典型应用、与自我效能感/社会认知理论区别
  是否需要对照页/导航页：不必单独导航，但应挂入“动机理论专题”
  需要补的鉴别诊断或相关条目：社会认知理论、自我效能感、人本主义心理学、习得性无助

  ### 3. PTG

  推荐标题：PTG 是什么？创伤后成长的五个维度、形成机制与常见误解
  推荐 description：解释 PTG（创伤后成长）的定义、五个维度、与韧性和症状减轻的区别，并澄清“创伤后成长不等于必须乐观”。
  推荐 synonyms：PTG、创伤后成长、创伤后积极变化、post-traumatic growth、创伤后成长五个维度
  推荐 H2 结构：PTG是什么、五个维度、如何发生、与韧性区别、与有毒正能量区别、临床与自助边界
  是否需要对照页/导航页：建议做 PTG / 韧性 / 症状缓解 区别表
  需要补的鉴别诊断或相关条目：PTSD、CPTSD、三阶段创伤治疗模型、侵入性反刍

  ### 4. 闪回

  推荐标题：闪回（Flashback）：创伤闪回、情感闪回与侵入性记忆的区别
  推荐 description：说明闪回是什么、为什么会发生、与侵入性思维和侵入性记忆有什么不同，并提供接地与求助建议。
  推荐 synonyms：闪回、创伤闪回、创伤性闪回、记忆闪回、PTSD闪回、flashback
  推荐 H2 结构：闪回是什么、常见类型、与侵入性思维/记忆区别、与解离区别、如何应对、何时需要专业帮助
  是否需要对照页/导航页：需要新增 侵入性体验区别页
  需要补的鉴别诊断或相关条目：侵入性思维、侵入性记忆、情感闪回与躯体闪回、接地、可耐受窗口

  ### 5. Tulpa 主词条

  推荐标题：Tulpa / 图帕 / 托帕：定义、创造流程、与多意识体/幻想伙伴的区别
  推荐 description：解释 Tulpa 是什么、如何被社群理解、创造流程有哪些阶段，以及它与多意识体、幻想伙伴、系魂的区别。
  推荐 synonyms：Tulpa、图帕、托帕、tulpamancy、tulpa是什么、tulpa创造、tulpa塑造、tulpa成声
  推荐 H2 结构：Tulpa是什么、起源与现代社群定义、创造流程、成声/附体/互动、与多意识体/幻想伙伴/系魂区别、风险与边界
  是否需要对照页/导航页：需要新增 多意识体 / Tulpa / 系魂 / 幻想伙伴区别
  需要补的鉴别诊断或相关条目：多意识体系统、幻想伙伴、Soulbond、Alterhuman、附体、Fronting

  ### 6. 解离量表导航页

  推荐标题：解离量表怎么选？DES-II、MID-60、SDQ-20 的用途、区别与解读
  推荐 description：用中文说明解离筛查常见量表的用途、适用对象、计分方式和解读边界，帮助用户理解“解离测试”不等于正式诊断。
  推荐 synonyms：解离量表、解离测试、DES、DES-II、MID-60、SDQ-20、解离体验量表、解离经验量表
  推荐 H2 结构：解离量表总览、DES-II适合谁、MID-60适合谁、SDQ-20适合谁、高分代表什么、不能替代诊断
  是否需要对照页/导航页：必须做，且应成为量表入口页
  需要补的鉴别诊断或相关条目：DID、OSDD、DPDR、解离性失忆、DSM-5-TR评估量表总览

  ### 7. 人格面具

  推荐标题：人格面具（Persona）：荣格概念、社交面具与 Masking 的区别
  推荐 description：解释人格面具在分析心理学中的原义，以及它在多意识体语境中的借用方式，并区分 Persona、社交面具和 Masking。
  推荐 synonyms：人格面具、Persona、社交面具、persona心理学、人格面具科普
  推荐 H2 结构：人格面具是什么、荣格原义、在多意识体语境中的使用、与Masking区别、与角色扮演区别、相关风险
  是否需要对照页/导航页：建议做 Persona / Masking / 社交伪装 区别表
  需要补的鉴别诊断或相关条目：Masking、荣格/分析心理学、人格结构理论、自我概念

  ## 6. 异常数据与清洗说明

  本次判断以 impressions >= 5 的关键词为主。impressions < 5 的词统一标记为“观察用，不作为主决策依据”。

  | 清洗问题                     | 例子                                                                | 处理原则                                         |
  | ---------------------------- | ------------------------------------------------------------------- | ------------------------------------------------ |
  | CTR 明显异常                 | cptsd 病例 200%、mid-60 200%、sdq-20 150%、延长暴露疗法 250%        | 视为统计异常或去重问题，不用于判断页面质量       |
  | 低样本词                     | 大量 1-4 impressions 的长句词                                       | 只用来发现新意图，不参与优先级排序               |
  | URL / 脏数据 / 乱码          | http：//wiki.mpsteam.cn、#NAME?、异常引号词                         | 单独归到“品牌/脏词”组，不做页面主题依据          |
  | 中英混杂 / 空格 / 全半角变体 | 抑郁自评量表 sds、self-determination theory 中文、焦 虑 自 评 量 表 | 统一归并到主关键词簇                             |
  | 繁简混杂                     | 解離經驗量表、莉莉絲                                                | 统一归并，不拆新页                               |
  | 同义词被拆散                 | 解离经验量表、解离体验量表、分离体验量表                            | 统一指向 DES-II 主页                             |
  | 临床误称 / 旧称              | 解离性人格障碍、双重人格、split personality                         | 作为主页 FAQ 与 synonym 处理，不单独做权威主词条 |
  | 非正式社群标签               | OSDD-1b、多意识体综合征                                             | 保留解释，但必须注明“非正式/非标准”              |

  一句话结论：短期最值得做的不是“再铺更多正文”，而是把现有高价值词条重构成更像搜索入口的页面体系，优先补 对照页 + 导航页 + 量表模板 + 误称归一。这样比单纯新增长文更快见
  效。
