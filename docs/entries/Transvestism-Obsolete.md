# 异装癖（Transvestism，已过时）

!!! warning "历史术语"
    “异装癖（Transvestism）”为 20 世纪常见的旧称，在不同分类系统中曾指代两类概念：

    - “恋物性易装癖”（ICD‑10：F65.1）：以穿着特定性别服饰获取性兴奋/满足的恋物性行为模式
    - “双重角色易装”（ICD‑10：F64.1）：以异性社会角色出现，但不以性兴奋为核心、也非跨性别身份

    DSM‑5‑TR 使用 “Transvestic Disorder”（易装障碍，302.3；ICD‑10‑CM 映射 F65.1）作为现代诊断，前提是伴随临床显著的痛苦或功能损害。ICD‑11 已删除上述 ICD‑10 旧项。该术语含污名且概念混杂，现已不推荐使用，应以更精确、非病理化的术语与框架表述。

!!! info "免责声明"
    本词条用于解读旧病历/旧文献中的历史术语与编码迁移；不构成诊断依据。如需评估与治疗，请咨询持证专业人员。

!!! info "Clinician’s Summary｜临床人员速览"
    “异装癖（Transvestism）”为历史术语，ICD‑10 下分 F65.1（恋物性易装癖）与 F64.1（双重角色易装）。DSM‑5‑TR 采用 “Transvestic Disorder”（302.3；ICD‑10‑CM F65.1 映射）并要求反复、强烈且持续的相关性兴奋意象/冲动/行为，同时伴临床显著痛苦或社会/职业功能损害才可诊断；一般的穿衣偏好或文化语境“变装”（drag）不构成诊断。ICD‑11 删除了 F65.1 与 F64.1，相关情形仅在满足痛苦/损害或风险前提时，才可能归入 6D36 “涉及独处或双方自愿的性行为之性偏好障碍”。与跨性别身份/表达或 [性别不安（Gender Dysphoria，GD）](Gender-Dysphoria-GD.md) 不等同；WHO 已将“性别不一致”移出精神障碍章节以去污名化。

## 旧称—现代分类（对照表）

| 体系 | ICD‑10 旧项/代码 | 现代对应 | 说明 |
|------|------------------|----------|------|
| ICD‑10 | 恋物性易装癖（F65.1） | ICD‑11：不再单列 | ICD‑11 删除该项；仅在符合 6D36 等前提（痛苦/损害或风险）时纳入相关“性偏好障碍”类别 |
| ICD‑10 | 双重角色易装（F64.1） | ICD‑11：删除 | 不等同跨性别身份；已取消该类别 |
| DSM‑5‑TR | Transvestic Disorder（302.3；ICD‑10‑CM F65.1 映射） | 仍保留 | 归入 Paraphilic Disorders；诊断须伴“临床显著痛苦/功能损害” |
| ICD‑11（参照） | 6D36 “涉及独处或双方自愿的性行为之性偏好障碍” | 现行分类 | 仅在存在痛苦/损害或风险时考虑；否则不归为疾病 |

> 注：ICD‑11 的“性别不一致（Gender Incongruence）”位于“性健康相关状况”章节（HA60/HA61），已移出精神障碍范围，用以减少污名化，与“易装/穿衣偏好”无直接对应。

## 现代分类如何处理

- ICD‑11（2019 发布，2022 生效）
    - 删除 ICD‑10 的 “F65.1 恋物性易装癖” 与 “F64.1 双重角色易装”。
    - 仅当行为造成显著痛苦/功能损害，或存在风险/非自愿情形时，才可能纳入“性偏好障碍”相关条目（如 6D36）。
    - 参考：ICD‑11 6D36（Paraphilic disorder involving solitary or mutually consensual sexual activity）、HA60/HA61（Gender Incongruence）。

- DSM‑5 / DSM‑5‑TR（2013/2022）
    - 使用 “Transvestic Disorder”（易装障碍，302.3；ICD‑10‑CM 映射 F65.1）作为 Paraphilic Disorders 的一类。
    - 诊断前提：反复、强烈且持续的相关性兴奋意象/冲动/行为，且伴临床显著痛苦或社会/职业功能损害。
    - 与“跨性别身份/性别表达”无直接等同关系；切勿将一般的跨性别穿衣或表演性变装（drag）病理化。

## 常见混淆与正确区分

- 性别身份 vs. 穿衣/表演
    - 跨性别身份相关的困扰与照护：参见 [性别不安（Gender Dysphoria，GD）](Gender-Dysphoria-GD.md) 及 [其他特定/未特定性别不安](Other-Specified-Unspecified-Gender-Dysphoria.md)。ICD‑11 将“性别不一致（Gender Incongruence）”移至“性健康相关状况”（HA60/HA61），并非精神障碍。
    - 文化/艺术语境中的“变装”（drag）属于性别表达或表演风格，本身并非临床诊断；与 DSM‑5‑TR 的 Transvestic Disorder 不等同。

- ICD‑10 旧称在现代的对照
    - F65.1 “恋物性易装癖” → ICD‑11 不再单列；在 DSM‑5‑TR 语境下，仅当符合“易装障碍”且有临床显著痛苦/损害时才考虑诊断。
    - F64.1 “双重角色易装” → 不等同于跨性别身份；现代更强调以身份、痛苦与功能影响为核心维度进行评估。

## 临床与沟通建议

- 用词建议
    - 面向公众与病历沟通：避免使用“异装癖”等带有病理化与污名的旧称；
    - 根据语境使用更准确的现代术语：如“性别不安”“性别不一致”“（表演性）变装”“穿衣偏好/性偏好”等。

- 评估重点（如涉及就医）
    - 痛苦/功能：是否存在临床显著的痛苦、强迫性冲动或功能损害（DSM‑5‑TR 前提）。
    - 同意性与风险：是否涉及非自愿、未成年人、伤害风险或其他安全议题（ICD‑11 纳入关键）。
    - 共病筛查：是否与性别身份困扰（GD/性别不一致）相关，或与其他障碍（如强迫、焦虑、物质使用、创伤）交织。
    - 去污名沟通：尊重身份与表达，采用创伤知情用语，避免将一般穿衣偏好或文化表演病理化。

## 病历解读与代码迁移（示例）

- 旧病历：ICD‑10 F65.1（恋物性易装癖）
    - 若现状符合 DSM‑5‑TR Transvestic Disorder 且伴临床显著痛苦/功能损害，可用 302.3/（ICD‑10‑CM F65.1 对应）说明；
    - 若仅为偏好/表演且无痛苦/损害，则不构成诊断，提供教育与支持性沟通更为合适。

- 旧病历：ICD‑10 F64.1（双重角色易装）
    - ICD‑11 已删除该项；
    - 建议评估是否存在 [性别不安（GD）](Gender-Dysphoria-GD.md)/ICD‑11 性别不一致（HA60/HA61）的照护需求，或仅为性别表达差异。

## 常见问答（FAQ）

- 喜欢异装是否意味着跨性别或需要就医？
    - 否。穿衣偏好与性别身份不同；仅当出现显著痛苦/功能损害或风险议题时，才需考虑临床评估。

- drag（表演性变装）会被诊断为障碍吗？
    - 不会。drag 属于文化/表演语境，与 DSM‑5‑TR 的 Transvestic Disorder 不等同。

- 何时应寻求专业评估？
    - 当相关行为导致显著痛苦、功能受损，或涉及非自愿/安全风险、与其他心理症状交织时，建议就医评估。
