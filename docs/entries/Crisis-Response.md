# 危机指南（Crisis Guide）

!!! warning "重要声明"
    本文仅提供通用的安全应对与自助方向，不替代专业诊疗或危机干预。
    若你或他人存在 **人身危险**、无法保证安全、或出现强烈自伤/自杀冲动，请优先联系当地紧急服务并前往就近医疗机构；避免独处。
    可用的热线与资源清单请参见《[危机与支援资源（Crisis & Support Resources）](Crisis-And-Support-Resources.md)》。

---

## 立即行动清单（30–180 秒）

> 目标：先把风险降到最低，再谈“理解发生了什么”。

1. **确保环境安全**：离开高处/水边/道路等危险区域；把自己移动到更安全、有人可见的空间。
2. **稳定身体与呼吸**：坐下或靠墙站稳，双脚踩地；把呼气放慢（例如“吸气更短、呼气更长”）。
3. **呼叫支持**：联系可信赖的人/照护者/治疗师；若无法联系到支持者且风险升高，立即寻求专业与急救支持。

!!! tip "如果你是多意识体系统"
    当下优先执行“最小安全动作”，尽量避免强制切换或争夺前台：

    - 用一句话做内部广播：“现在需要安全与稳定，先做三步清单。”
    - 若有守门人/保护者/照护者角色，邀请其协助维持环境安全与对外沟通。

## 快速接地（选 1–3 个即可）

- **5-4-3-2-1**：说出 5 样你看见的东西、4 种你能触到的触感、3 个声音、2 种气味、1 种味道。
- **自我定向**：说出“我是谁、我在哪里、今天是几号、现在更接近白天还是夜晚”。
- **身体锚定**：把注意力放在脚底与地面的接触点；轻压大腿或双臂，感受肌肉用力与放松。
- **环境线索**：找 3 个“现在才会出现”的事物（例如房间布置、手机锁屏时间、窗外天气）来提醒自己“此刻在当下”。

想系统学习可见： [接地（Grounding）](Grounding.md) 与 [自我照护工具箱](Self-Care-Toolkit.md)。

## 常见问题（FAQ）

??? question "我出现强烈解离、空白感或“不真实感”，怎样尽快稳定？"

    - 优先完成「立即行动清单」，把自己移动到更安全、有人可见的地方。
    - 选 1 个接地方法做到“能重复 3 轮”，而不是一次做很多种。
    - 若出现明显的失控风险、无法自我照护、或频繁“失去时间”，尽快联系专业支持，并把当下状态简要告知支持者。

??? question "惊恐发作时我很害怕，是不是要立刻去医院？"

    - 惊恐发作本身通常会在一段时间内回落；关键是 **先确保安全** 并稳定呼吸与身体姿势。
    - 如果你出现胸痛、呼吸困难、昏厥前兆、或任何你担心的躯体急症信号，请优先就医排除身体风险。
    - 参考： [惊恐发作（Panic Attack）](Panic-Attack.md)。

??? question "强烈闪回/噩梦后我“回不来”，可以怎么复位？"

    - 先做接地与自我定向：把注意力拉回房间、光线、温度与当下时间。
    - 做一个“低成本的现实动作”：喝水、洗手、换一件衣物、打开灯或拉开窗帘，让身体获得“现在”的线索。
    - 若你发现自己持续处在高唤醒/冻结状态，建议把这次经历记录下来（见下方「简短记录」）并在安全时寻求支持。
    - 参考： [创伤性闪回（Flashback）](Flashback.md)。

??? question "出现自伤/自杀意念时，我该怎么做？"

    - **立刻把风险降到最低**：远离可能造成伤害的环境与物品；尽量不要独处，移动到更安全、有人可见的空间。
    - **联系支持与专业帮助**：尽快联系可信赖的人、治疗师或当地危机/急救资源；如果你担心自己无法保持安全，请直接寻求紧急帮助。
    - **使用危机卡**：把“对我有效的最小动作”和“该联系谁”按清单执行（见下方模板）。
    - 若你正在当下处于危险中，可直接跳转至《[危机与支援资源（Crisis & Support Resources）](Crisis-And-Support-Resources.md)》获取进一步求助方式。

## 系统/多意识体特有注意

### 前台协调与交接（优先级：安全 > 清晰 > 解释）

- **避免强制切换**：在高压时强推切换可能加重混乱；优先让“最能保持安全”的成员协助稳定。
- **使用安全词/信号**：提前约定一个简单信号（词语/手势/短句），含义是“停止争执，执行最小安全动作”。
- **降低信息负载**：对内对外沟通都尽量短句化；一次只处理一个问题。

### 与保护者/照护者的快速沟通清单

- 现在最大的风险是什么？（环境/冲动/失控/外界压力）
- 哪 1–2 个动作最可能立刻降低风险？（移动到安全处、联系某人、接地）
- 需要谁来对外沟通？需要谁来“守住前台安全”？

### 简短内部广播与记录（用于事后复盘）

建议在可行时记录 4 项即可：

- **时间**：大概几点（不必精确）
- **地点**：在家/学校/户外/医院等
- **状态**：解离/恐慌/闪回/强烈冲动（可多选）
- **已执行**：做了哪些最小安全动作？是否已联系支持者？

参考： [内部沟通（Internal Communication）](Internal-Communication.md)。

## 危机“随身卡”与安全计划（模板）

!!! tip "个人危机卡（可打印一页）"

    - 我的预警信号（出现这些说明需要启动危机流程）：____
    - 对我最有效的 3 个“最小动作”（越简单越好）：____ /____/____
    - 我可以联系的人（按优先级）：____（关系/方式）→____→____
    - 我可以去的安全地点：____
    - 我希望他人怎么帮助我（短句）：____
    - 若我无法保证安全：____（例如“立刻寻求紧急帮助并陪同就医”）

!!! info "陪伴者/同伴简表（1 分钟版）"

    - 保持在安全空间与可见范围，语速放慢、句子变短。
    - 先问一个问题：“现在最需要的是 **安全**、**呼吸** 还是 **联系谁**？”
    - 避免争辩细节、追问原因或强迫回忆；优先执行最小安全动作。
    - 若风险升级或无法保证安全，协助联系专业支持并陪同就医。

## 何时必须线下就医（红旗信号）

- 无法保证自身或他人安全，或出现强烈自伤/自杀冲动。
- 严重失控、意识混乱、频繁“失去时间”，无法完成基本自我照护。
- 出现疑似急症的躯体信号（例如明显胸痛、呼吸困难、昏厥前兆等）或你对自身状况高度不确定。
