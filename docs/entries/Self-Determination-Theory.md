# 自我决定理论（Self-Determination Theory）

## 概述

自我决定理论由德西与瑞安提出，强调自主性、胜任感与关系感三大基本心理需求。当这三项需求得到满足时，个体更能维持内在动机与心理健康。

!!! tip "快速答案"
    **自我决定理论（Self-Determination Theory, SDT）** 是解释动机质量的心理学理论。它关注人是否感到“我能选择”“我能做到”“我被连接”，而不只是外在奖励或惩罚是否足够强。

## 常见问题（FAQ）

??? question "自我决定理论是什么？"
    自我决定理论（Self-Determination Theory, SDT）是德西（Deci）和瑞安（Ryan）提出的动机理论，用来解释一个人为什么会主动投入、持续坚持，或在高压下逐渐失去动力。它强调，动机质量和心理健康与环境是否支持个体的基本心理需求密切相关。

??? question "自我决定理论的三种基本心理需求是什么？"
    最核心的三项是：

    - **自主性**：我是否感觉自己有选择权，而不是被强迫。
    - **胜任感**：我是否觉得自己能做到、能进步、能产生效果。
    - **关系感**：我是否感到被理解、被接纳、与他人有联结。

    很多人搜索“自我决定理论 三种基本心理需求”，找的就是这三项。

??? question "self-determination theory 和 SDT 是同一个吗？"
    是的，`self-determination theory`、`SDT`、`自我决定理论` 指的是同一理论。中文语境里有时也会搜“德西自我决定理论”或“德西的自我决定理论”，本质上都是在找 Deci 与 Ryan 这套动机框架。

??? question "自我决定理论是谁提出的？"
    自我决定理论由 Edward L. Deci 与 Richard M. Ryan 系统提出并发展。早期研究关注内在动机，后来扩展为包含基本心理需求、动机连续体、环境支持和幸福感的综合理论。

??? question "自我决定理论可以用在情绪干预或系统协作里吗？"
    可以作为沟通和环境设计的参考，但不能替代临床治疗。它的实用价值在于提醒支持者减少控制式要求，增加选择、可完成目标和稳定连接，从而降低抵抗与耗竭。

## 核心概念

- **动机连续体** : 从无动机到外在调节、内摄调节、认同调节直至内在动机。
- **基本心理需求** : 自主性、胜任感、关系感是普遍需求，跨文化适用。
- **支持性环境** : 提供选择、反馈与连结可促进自主性动机。

## 理论发展

自我决定理论应用于教育、体育、健康、组织管理等领域，并衍生出动机式访谈等实践。最新研究关注数字环境与远程工作的需求支持策略。

## 在多意识体与解离语境中的意义

系统成员若感受到自主权、能力与连结，更容易协作并保持动力。反之，控制式管理容易引发抵抗或[认知失调](Cognitive-Dissonance.md)。因此，系统治理可借鉴 SDT 来设计公平、尊重的流程。

## 实践应用与建议

- 在内部决策中提供选择，让成员表达偏好与界限。
- 设定可以掌握的阶段目标，提供及时肯定，强化胜任感。
- 建立支持网络，让成员感受到被理解与连结。
