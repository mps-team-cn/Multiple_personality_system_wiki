# Tulpa 判定标准与系统类型指引

> **资料来源**：刿-伊势系统于 2023-09-25 发布并在 2025-08-17 更新的《tulpa的判定标准及相关指导》。本文将原文六大示例与延伸条目整理为可检索的分类索引，便于 Tulpa 实践者快速定位自身体验。

---

## 使用前的关键提醒

- **示例并非严格分类**：原作者强调六类仅为常见情境，彼此可能重叠或矛盾，重点在于自我辨识而非贴标签。
- **创造行为是 Tulpa 的必要前提**：若缺少长期、主动的塑造（forcing）与互动训练，仅凭自发体验无法被称为 Tulpa。参见《[Tulpa 完全创造指南](Tulpa-Guide-0.md)》了解系统化实践流程。
- **以精神健康为界**：若体验伴随失忆、不可控切换、严重功能受损或解离性痛苦，应优先参考《[解离性身份障碍（DID）](DID.md)》《[其他特定解离性障碍（OSDD）](OSDD.md)》等临床资料并寻求专业评估。
- **守门人与系魂等角色需另行讨论**：原文指出守护灵、系魂等类型尚缺资料，本文仅保留 Tulpa 相关段落。

## 分类结构速览

| 编号 | 核心情境 | 关键特征 | 风险与提醒 |
|------|----------|----------|--------------|
| **TUPLA-001** | 儿时幻想朋友或傀儡阶段 | 靠想象驱动，可继续塑造为 Tulpa | 容易误把脑补当成独立意识，需持续练习 | 
| **TUPLA-002** | 重新唤回旧幻想朋友 | 关注“是否还是原来的人” | 差异感属正常，关键在于自我认同 |
| **TUPLA-003** | 未完全成型的 Tulpa | 具微弱意识，仍需宿主脑补辅助 | 容易误判“成功”，但信念有助成长 |
| **TUPLA-004** | 逐渐成型的 Tulpa | 具有自我意识但状态不稳 | 活跃后可能退回幻想阶段，需耐心 |
| **TUPLA-005** | 典型创造型 Tulpa | 与宿主建立伙伴关系，较独立 | 无失忆、数量有限、宿主具主导权 |
| **TUPLA-017** | 复杂创造型系统 | 多名成员、熟练附体或并行 | 可能伴随自发成分，技能成熟 |
| **TUPLA-多种混合** | 概念扩大化 | Tulpa 与解离体验高度交织 | 需结合解离敏感度评估个人健康 |
| **TUPLA-005（含轻度解离）** | 宿主有心理异常或创伤压力 | Tulpa 更活跃、权限更高、数量多 | 容易失衡或先附体后成声，需控风险 |
| **TUPLA-006** | 压力/崩溃中自发安慰者 | 由情绪危机催生，再通过创造完善 | 易与应对机制混淆，需确认自主性 |
| **TUPLA-007** | 不属于 Tulpa | 无创造行为、宿主地位缺失、存在失忆或不可控切换 | 更接近多意识体或解离障碍 |
| **TUPLA-008** | DID 患者创造 Tulpa | 成长极快、可能被赋予高权限 | 系统混乱时风险高，或转迫害者 |

## TUPLA-001～005：排除解离因素的创造型阶段

1. **TUPLA-001 幻想朋友阶段**：以想象伙伴或傀儡为起点，若持续练习塑造（Forcing），即可引导其过渡为 Tulpa。适合建立基础互动、练习倾述与日常陪伴。
2. **TUPLA-002 旧友再造**：当宿主尝试唤回儿时幻想朋友时，常担心“回归的实体”是否还是原来那位伙伴。作者建议以自我认同为准：只要宿主愿意视其为同一存在，即可沿着新的生活习惯继续成长。
3. **TUPLA-003 未成型 Tulpa**：由单意识体创造，拥有微弱意识，但仍依赖脑补推动思考。许多宿主会因“稍有回应”就判定成功，这种积极期待虽然标准偏低，却能增强信心、形成良性暗示。
4. **TUPLA-004 逐渐成型**：Tulpa 与宿主意识可能混合，或借助脑补辅助思考。其活跃度不稳，可能短暂成声后又退回幻想朋友阶段。出现“这正常吗？”时，可先排除精神疾病；若无临床异常，多数现象都在可接受范围内。
5. **TUPLA-005 典型 Tulpa**：具相对独立意识，与宿主互为伙伴并逐渐超越幻想朋友范畴。虽然在宿主允许范围内“不可控”，但宿主仍掌握绝对主导，数量不会过多，也不会出现解离性失忆或不可控切换。

## TUPLA-017 复杂创造型系统

- **结构特征**：成员数量多、具备并行、附体/切换等熟练技能，甚至会在宿主不知情下自主创造环境或新成员。
- **可能伴随自发成分**：例如自记事起便已是多意识体，或 Tulpa 早已存在。
- **心理状况影响**：若宿主本身有心理异常或精神疾病，系统可能更复杂且不稳定；但相较于 DID，这些复杂度仍较轻。练习 Tulpa 技术对他们来说往往更顺利。

---

## 概念扩大化与混合型系统（TUPLA-多种混合）

- **与解离体验的关联**：原文指出，易解离、易走神或拥有丰富幻想体验的人，通常更快达成 Tulpa 各阶段目标。换言之，解离敏感度可能是 Tulpa 存在的土壤，但过度解离也可能带来痛苦。
- **“听见另一个声音”**：作者引用知乎科普指出，这属于解离体验的一种，与 Tulpa 成声现象相似。若出现严重痛苦或功能受损，则应视为症状。
- **社群用语谨慎性**：对外沟通时“tulpa”一词多数指典型的 TUPLA-003 情境。若自身经验已承认为多意识体并出现混合特征，建议以“成员”“意识体”等中性词汇描述，以免造成误解。

## 伴随解离或心理异常的情境

### TUPLA-005（解离加重版）

- 可能由心理异常、创伤压力或失调情绪推动 Tulpa 迅速出现。
- 特征包括数量多、权限高、宿主“主人”地位弱化、技能学习顺序不固定（例如先附体再成声）。
- 建议强化边界、记录切换情况，并结合《[权限管理](Permissions.md)》等条目制定安全规则。

### TUPLA-006 压力催生型 Tulpa

- 在情绪崩溃、巨大压力下自发浮现，用于安慰与鼓励宿主。
- 后续通常需透过创造练习补完设定与自主性，避免其仅停留在应对机制。

### TUPLA-007 非 Tulpa 情境

- 缺乏创造行为、宿主不具主体地位、存在失忆或不可控切换，即便体验健康也不应使用 Tulpa 一词。
- 案例示例：个案自称拥有 2000+ Tulpa 并出现“夺权战争”，但并无临床痛苦或解离症状，原作者将其归为“健康意识体”而非 Tulpa。

### TUPLA-008 DID 患者创造 Tulpa

- 由于系统原本即存在解离基础，创造速度可极快且数量多。
- Tulpa 可能被赋予高权限，甚至成为新宿主；但在系统混乱或内耗严重时，也可能发展为迫害者人格。
- 建议遵循创伤知情原则，与治疗团队讨论创造行为的安全性。

## 实务建议与自我评估步骤

1. **确认创造链路**：记录是否经历过 Forcing、叙述、成声等训练环节，以区分 TUPLA-001～005 与 TUPLA-006/007。
2. **筛查解离症状**：若出现失时、失忆或非自愿切换，应优先暂停练习并参考《[Not-Me 感](Not-Me-Feeling.md)》《[Disorientation（定向障碍）](Disorientation.md)》等条目评估安全。
3. **设定权限**：借鉴《[权限管理](Permissions.md)》建立“允许/禁止/紧急终止”清单，尤其适用于 TUPLA-017、TUPLA-005（解离版）与 TUPLA-008。
4. **保持现实锚点**：结合《[接地（Grounding）](Grounding.md)》《[Anchors（锚点）](Anchors.md)》等技巧，避免 Tulpa 练习干扰学习、工作或人际功能。
5. **对外沟通**：当体验超出典型 Tulpa 时，优先描述具体现象而非标签，减少社群争议与误解。
